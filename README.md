# Import-Rreconstructor
Sample program to dump and automatically restore the import table

1. Дампится загруженный образ, в нём финсится заголовки секций. Секция в дампе становится такой же, как и в памяти. PointerToRawData приравнивается к VirtualAddress.
2. Определяются все используемые динамические библиотеки
3. Партится таблица экспорта у всех библоитек из списка и определяются адреса всех их экспортируемых функций
4. Партится таблица импорта, по адресу функции определяется её имя (п.3)
5. Дописываются имена функций

Программка демонстрационная.
